Drive'inchi - Firebase Web MVP

1. Replace the Firebase config in `app.js` with your actual Firebase Web App config
   (get it from Project Settings > Web App > SDK setup)

2. Upload all these files using:
   Firebase Console > Hosting > "Manually upload files"

3. Visit your public URL to test it live

- Supports selecting car model, color, wheels, headlights
- Saves customization to Firestore under `car_customizations` collection
- Loads saved data by document ID
